source insight入门教程

作者：eclipsa

摘要：描述source insight的一些基本概念和用法。

[toc]

什么是source insight

简单来说，是一个面向软件开发项目的代码阅读/编辑器。高性能(大型项目的适用性)，上下文分析/提示，函数跳转，是它最重要的特点。

关键概念

project

project，指项目。而软件的项目指的是完成一个指定目标的全部代码。source insight是面向项目开发，针对项目进行管理。

source insight的每个项目有两个项目文件夹地址。一个是源码文件夹，也就是项目源代码。另外一个是source insight关于该项目管理的文件夹，存储诸如分析过的符号信息、源码文件树、编辑临时文件之类的内容。

工作流程

新建project

用Project > New Project命令创建project。

(project配置中两个快速搜索功能默认启用，会提高硬盘空间和内存的占用程度。特大项目看情况避免卡顿。)

为project添加文件

在工具栏中Project > Add and Remove Project Files选项下，可以执行，添加/移除文件。

打开project

在工具栏中Project选项下，可以执行，打开一个project。

编辑内容

开发代码。

操作详解

项目文件

Project > Project Settings中支持Master File List，创建项目文件树配置文件。可以放在项目中，其他开发人员就可以轻松同步项目文件树状态。

配置文件格式如下

;注释

FM15Output\*.bat

操作project

添加/删除project

在工具栏中Project选项下，可以执行打开、关闭、删除、复制、设置一个project。

(对应英文new、close、remove、copy、project setttings。)

同步符号表

在工具栏中Project > Synchronize Files选项下，可以执行同步符号表。适用于不在source insight中修改过代码时。

重建项目

在工具栏中Project > Rebuild Project选项下，可以重建项目，适用于source insight项目异常时修复之用。重建source insight的项目管理相关文件，非源码文件。

操作project文件

添加/移除

在工具栏中Project > Add and Remove Project Files选项下，可以执行，添加/移除文件。

(移除只是从项目中移除，不是删除。)

新建文件

ctrl n用于新建文件。

格式化换行符

Edit > Special Edit > Normalize End Of Line Characters中可以修改当前文件的换行符。

Options > Peference > Files > Default line ending中可以修改默认文件的换行符，用于新建文件。

扩展操作模块

关系窗口

View > panel > Relation Window用于显示关系窗口，relation window。可以显示某个语法元素与其相关的所有其他语法元素关系，如函数调用关系，类关系。关系窗口支持多种多种视图模式，可以在上面用右键调整。

上下文窗口

View > Panel > Context window用于显示上下文窗口，context window简单的说就是一个自动预览相关信息的窗口，诸如函数定义，变量定义。

代码段窗口

View > Panel > Snippet Window，用于管理自定义的常用代码片段，供快速输入用。如通用的while循环结构，某种语言专用的for循环结构。

搜索栏

view > File Search Bar用于启用搜索栏，适用于搜索当前打开的文件。

视图操作

上移

alt ↑用于向上移动代码视图窗口。

下移

alt ↓用于向下移动代码视图窗口。

左移

alt ←用于向左移动代码视图窗口。

右移

alt →用于向右移动代码视图窗口。

编辑操作

跳转到定义

  • 在语法元素上ctrl click

  • 选中语法元素后ctrl =

  • 选中语法元素后Search > Jump to Definition

可以实现跳转到定义，Jump to Definition。

跳转到调用

ctrl alt c可以实现跳转到调用，Jump to Caller。也可以在定义上右键Jump to Caller。

文本搜索

ctrl f在当前文件中进行搜索，支持正则匹配。也可以用Search > Search。

文本向下/向上搜索

F4或者F3，向下/向上执行文本搜索或引用搜索。也可以用Search > Search Forward或者Search > Search Backward。

项目文件搜索

Search > Search Project在项目中进行搜索，支持正则匹配。

引用搜索

ctrl /此命令针对语法元素，可以搜索全项目对该语法元素的引用，使用项目分析的缓存文件，速度较快。也可以用Search > Lookup References。

搜索并替换

ctrl h或Search > Replace可以搜索并替换本文件的部分文本内容，支持正则。也可以用Search > Replace。

多文件搜索并替换

ctrl shift h或Search > Replace Files可以搜索并替换本文件的部分文本内容，支持正则。也可以用Search > Replace Files。

智能重命名

ctrl '针对语法元素进行本地级别/项目级别的重命名替换。

自动缩进

source insight选中一些语法元素，按tab就可以自动缩进代码。

自动反向缩进

source insight选中一些语法元素，按shift tab自动反向缩进代码。

自动注释

Edit > Special Edit > Comment Lines自动单行注释。

自动取消注释

Edit > Special Edit > Un-Comment Lines自动取消单行注释。

撤销

ctrl z用于回滚上次编辑的操作。也可以用Edit > Undo。

重做

ctrl y用于重执行回滚的编辑的操作。也可以用Edit > Eedo。

选中行

鼠标于同行尾部外，左键双击可以选中一行。

选中且复制当前行

ctrl k用于选中及复制当前光标所在行。

移动与选择

预览窗口

推荐的文本移动方式，直接点击预览窗口配合鼠标滚轮。

按行跳转

ctrl g可以实现跳转到指定行。

闭合符号选择

双击闭合符号即可选中整个闭合块。闭合符号包括，括号、方括号、花括号、双引号。

闭合块选择

ctrl -在光标选中范围外选择闭合块(闭合方括号、花括号)进行内容选择。多次使用则，可以逐步选中不同量级的封闭块。

多行选择

alt 鼠标左键用于框选代码。适用于删除。

正则表达式

source insight支持两种正则表达式模式，分别是"Source Insight"和"Perl Compatible”。

Source Insight正则

基本词

语法 意义 匹配示例
. 匹配一个任意字符 b.g匹配bag和b1g
^ 匹配行首
$ 匹配行尾
[abc] 匹配一个范围内的字符 [abc]匹配a或c
^[abc] 匹配一个不在范围内的字符 ^[abc]匹配d
\t 匹配一个tab
\s 匹配一个空格字符
\w 匹配一个空白字符
\n 匹配换行符

基本词(基本组)

语法 意义 匹配示例
[abc] 匹配一个范围内的字符 [abc]匹配a或c
^[abc] 匹配一个不在范围内的字符 ^[abc]匹配d
[a-z]或[a-zA-Z]或[a-zA-Z0-9] 匹配一个范围内的字符

量词

量词是对前一个正则基本词的修饰，补充它的数量关系。

语法 意义 匹配示例
* 匹配0个及以上的内容
+ 匹配1个及以上的内容

正则分组

用\(和\)包裹一个正则基本词结构，代表一个匹配的一个分组。去掉\(和\)代表无分组。分组为二次提取内容，相当于完成匹配后进行多一个分组匹配并分离出分组的部分，适用于替换内容。

字符转义

\符号用于转义正则中用道的语义字符。

操作工具

文件对比

Tools > Compare File启用文件对比功能。

文件夹对比

Tools > Compare with Backup File启动当前文件与source insight的文件自动备份进行对比。

备份文件对比

Tools > Directory Compare启动文件夹(多文件)对比功能。面板仅列出文件状态。双击进入文件对比。

代码格式化

Tools > Reformat Source Code with xxx Preset可以格式化代码到某一种风格。风格在Tools > Reformat Source Code Options中调整。

重置设置

Options > Preferences > General中的Use Defaults可以重置source insight的配置回到默认状态。

个性化配置

Options > File Type Options中支持针对某种语言进行的个性化管理配置。

显示行号

Options > File Type Options中Show line numbers能够配置当前编程语言默认显示行号。

View > Line Numbers可以在当前文件显示行号。

显示右边界

Options > File Type Options中Show right margin能够配置当前编程语言默认右边界，用于辅助控制单行字符的长度。

Options > File Type Options中Margin width用于控制右边界显示的位置。

tab自动转spaces

Options > File Type Options中Expand tabs to spaces能够配置当前编程语言默认自动将tab转化为空格。

Options > File Type Options中Tab width用于控制tab转化空格的数量。

tab和spaces可视化

Options > File Type Options中Visible tabs和Visible spaces能够配置当前编程语言默认自动以•代替显示tab和space。

View > Visible Tabs and Spaces可以在当前文件自动以•代替显示tab和space。

换行符可视化

View > Visible End Of Line可以在当前文件以ASCII符号简称显示行号符。
