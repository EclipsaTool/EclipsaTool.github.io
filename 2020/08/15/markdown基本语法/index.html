Markdown基本语法

作者：eclipsa

摘要：本文旨在提供较为标准部分的Markdown语法指南，并不记录全部Markdown语法。

[toc]

什么是Markdown

Markdown是一种具有纯文本格式语法的轻量级标记语言。

目的：提供一种易于阅读和易于编写的纯文本格式书写，可选将其转换为的XHTML/HTML。

标准化：尚未完全标准化，故软件对其支持和扩展有所异同。

基础语法

标题

设置标题

Markdown HTML
# 一级标题 一级标题
## 二级标题 一级标题
### 三级标题 一级标题
#### 四级标题 一级标题
##### 五级标题 一级标题
###### 六级标题 六级标题

语法详解：用#开头，然后输入一个空格，然后输入标题内容。

语法意义：#的数量代表标题的级别。

设置标题IDs

注意：此语法支持并没有广泛支持。

Markdown HTML
# 一级标题{#标题id} 一级标题

语法详解：在标题之后输入一个{}，然后在其中输入#，然后输入自定义标题id。

语法意义：标题id可用来作为锚点，使超链接可以跳转到相应的地方。

段落

Markdown HTML
内容 空白行 内容 内容 空白行 内容

语法详解：用空白行分割的内容就会称为两个段落。

语法注意事项1：段落开头不应该有空格或tab。

断行

Markdown HTML
内容 空格 空格 回车 内容 内容内容

语法详解：在句子尾部输入两个以上的空格，然后换行就可以断行。

语法意义：这适用于同一个段落内需要断行的地方。断行前后的内容同属于一个段落

转义字符

Markdown
\|

语法详解：输入一个反斜杠后输入需要转义的字符即可将Markdown语法中用的一些字符转义为字符，从而可以显示。

强调

斜体

Markdown HTML 渲染效果
这是*斜体*内容 这是<em>斜体</em>内容 这是重点内容

语法详解：在内容前后输入1个*号即可以使内容加斜。

加粗

Markdown HTML 渲染效果
这是**粗体**内容 这是<strong>粗体</strong>内容 这是粗体内容

语法详解：在内容前后输入两个*号即可以使内容加粗。

粗斜体

Markdown HTML 渲染效果
这是***粗斜体***内容 这是<strong><em>粗斜体</em></strong>内容 这是粗斜体内容

语法详解：在内容前后输入3个*号即可以使内容加粗斜。

删除线

Markdown HTML 渲染效果
这是~~删除线~~内容 这是<del>删除线</del>内容 这是删除线内容

语法详解：在内容前后输入两个~号即可以使内容有删除线。

区块引用

Markdown
>这是区块引用的内容

语法详解：在每行前输入一个或多个>号，然后输入一个空格，即可以定义一个区块。每个>号增加块引用的层级一次。块引用支持连续多行合并，如同一个块一样。

语法意义：区块引用用于定义一个长引用文段。

语法注意事项1：区块引用支持多行合并，如果想在同一个区块内放两段引用文段，两段引用文段中间补上一个同等级的空区块引用即可。效果如下

>引用文1
>
>引用文2

语法注意事项2：区块引用的多层嵌套，效果如下

引用文1

引用文2

语法注意事项3：区块引用可以支持其他元素，如粗体，标题等，具体情况需要测试。

列表

有序列表

Markdown HTML
1. 内容 <ol><li>内容</li></ol>

语法详解：以阿拉伯数字开头，然后输入一个.号，然后输入一个空格，即定义了一个有序列表的一项。

语法注意事项1：有序列表支持多层嵌套。前面打n组四个空格，然后输入有序列表可以实现n级有序列表。

语法注意事项2：有序列表对数字的值没有要求，顺序数值大小随意。

语法注意事项3：有序列表可以支持其他元素，如粗体，标题等，具体情况需要测试。

无序列表

Markdown HTML
- 内容 <ul><li>内容</li></ul>

语法详解：以-号开头，然后输入一个空格，即定义了一个无序列表的一项。

语法注意事项1：无序列表支持多层嵌套。前面打n组四个空格，然后输入有序列表可以实现n级无序列表。

语法注意事项2：无序列表可以支持其他元素，如粗体，标题等，具体情况需要测试。

任务列表

功能 Markdown
未完成任务 - [ ] 任务描述
已完成任务 - [x] 任务描述

语法详解：以-号开头，然后输入一个空格，然后一个[]，[]中空格代表未完成，x代表已完成，随后输入任务描述。

自定义列表

注意：此语法支持并没有广泛支持。

语法如下

类别
: 种名

HTML如下

<dl>
    <dt>类别<dt>
    <dd>种名d<dd>
</dl>

语法详解：第一行定义类别名，第二行以:号开头，然后输入一个空格，接下来输入种名即可。

图片

Markdown HTML
![失败描述](/assets/images/tux.png) <img src="/assets/images/tux.png" alt="失败描述">
![失败描述](/assets/images/tux.png "图片标题") <img src="/assets/images/tux.png" alt="失败描述" title="图片标题"

语法详解：以!号开头，随后是[]，最后是()。[]中填写图片描述，(填写文件路径或者url)

语法注意事项1：失败描述在图片加载失败时可能会以文字形式出现(视渲染支持)。图片标题在图片加载后鼠标悬浮于图片之上一会会显示。

代码

单行代码

Markdown HTML
`代码行` <code>代码行</code>

语法详解：前后输入`号包裹代码即表示为代码，不尝试渲染其他元素。仅限一行以内的内容。

代码区块

语法如下

{
  "firstName": "John",
  "lastName": "Smith",
  "age": 25
}

效果

{
  "firstName": "John",
  "lastName": "Smith",
  "age": 25
}

语法详解：代码内容前后输入三个`号开头的空行，第一组后面可以跟代码的语言名称。渲染器可以用于成立代码高亮。

水平线

Markdown HTML
—

语法详解：以三个-号开头的空行将变成水平线。

语法注意事项1：三个-号前后不要有内容。

超链接

直接超链接

Markdown HTML
<test@example.com> <a href="test@example.com">test@example.com</a>
<http://test.com> <a href="http://test.com">http://test.com</a>

语法详解：将URLs或邮箱地址放在<>之中可以生成超链接。

语法注意事项1：简易超链接支持字体强调语法。

格式化超链接

Markdown HTML
[显示文本](http://test@example.com) <a href="http://test@example.com">显示文本</a>
[显示文本](#锚点名) <a href="#锚点名">显示文本</a>

语法详解：输入一个[]，然后输入一个()。在[]中输入要显示的文字，在()中输入超链接的指向。

语法注意事项1：格式化超链接支持字体强调语法。

引用式超链接

功能 Markdown HTML
引用超链接 [显示文本][超链接别名]
超链接源 [超链接别名]: http://test@example.com <a href="http://test@example.com">显示文本</a>
超链接源 [超链接别名]: http://test@example.com 链接标题 <a href="http://test@example.com" title="链接标题">显示文本</a>

语法详解：引用超链接语法是先输入两个[]，第一个放显示文本，第二个放超链接别名，链接别名用于绑定超链接源。设置链接源语法为先输入一个[]，在[]中输入超链接别名，然后输入:号，一个空格，随后是超链接。

语法目的：方便重复引用同一个超链接。

语法注意事项1：超链接别名不区分大小。

语法注意事项2：引用式超链接中超链接源可以随意放置在文章中任意行。一行一条。建议统一写在文末。

HTML

Markdown支持使用HTML的语法，但有部分元素不支持。在块级HTML中不能再使用Markdown语法。

扩展语法

表格

表格语法如下

|标题1|标题2|
|---|---:|
|内容|内容|
|内容|内容|

渲染效果

标题1 标题2
内容 内容
内容 内容

语法详解：用|号将内容包裹。第二行内容全部用-号定义表格标题栏，每个内容至少3个-号，在-号最左或者最右输入:号将使该栏支持左或右对齐，如果左右都有:号则为中间对齐。

语法注意事项1：表格内支持一些内嵌元素。包括链接、单行代码、强调等。不包括标题、多行元素、水平线、HTML标签等。

脚注

功能 Markdown
引用脚注 [^脚注别名]
设置脚注源 [脚注别名]: 注释内容

语法详解：引用脚注语法是先输入一个[]，在里面输入一个^号，然后是脚注别名，可以是数字。设置脚注源语法为先输入一个[]，在[]中输入一个^号和脚注别名，然后输入:号，一个空格，后面跟脚注内容。

语法目的：为文段添加注释或词汇添加注释。

语法注意事项1：超链接别名不区分大小。

表情

Markdown 渲染效果
😂 😂

语法详解：前后用:号包裹，中间输入表情英文简码即可。
